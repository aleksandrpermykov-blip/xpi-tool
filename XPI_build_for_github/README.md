# XPI — Подбор гильз и муфт (Raychem)

Десктоп-утилита для подбора обжимных гильз, муфт и инструмента
при соединении греющих кабелей серии XPI и XPI-S.

## Как получить exe

1. Загрузи этот репозиторий на GitHub
2. Перейди во вкладку **Actions** → **Build Windows EXE** → **Run workflow**
3. Через ~3 минуты скачай **XPI_Portable_Windows** из артефактов

## Запуск

Просто дважды кликни `XPI_Подбор_гильз.exe` — установка не нужна.

## Возможности

- Выбор кабеля 1 и кабеля 2 из выпадающих списков
- Автоматическая фильтрация совместимых кабелей
- Подбор гильзы, матрицы, инструмента и набора CS-150-xx-PI
- Подбор гильзы для оплётки
- Расчёт Вт/м и мощности цепи по заданным длинам и напряжению
